# micro-skills

一个轻量级的 Claude Code 技能集合，旨在提高开发效率和代码质量。

## 项目简介

micro-skills 收录了各种实用的小型技能，每个技能专注于解决特定的开发任务。这些技能遵循简洁、高效的设计原则，易于安装和使用。

## 可用技能

### ac (Auto Commit)

自动创建符合 Conventional Commits 规范的 Git 提交消息。支持智能分析暂存或未暂存的更改，自动检测不同类型的更改并生成规范的提交消息，也可以将大型提交拆分为多个原子提交。支持的提交类型包括：feat（新功能）、fix（Bug 修复）、docs（文档更改）、style（代码风格）、refactor（重构）、perf（性能改进）、test（测试）和 chore（维护任务）。使用时直接运行 `/ac` 即可，无需额外参数。

## 安装

### 使用 cc-switch (推荐)

1. 确保已安装 [cc-switch](https://github.com/farion1231/cc-switch)
2. 在界面中点击 "Skills" -> "发现技能" -> "仓库管理" -> "添加仓库 https://github.com/joizhang/micro-skills"
3. 搜索 "micro-skills" 并安装相应的技能
